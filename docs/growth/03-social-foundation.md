## First 10 posts (ready to publish)

### X / LinkedIn Post 1 — Thread opener (pin)

**X thread (6 tweets):**

1/ Most Coast FIRE calculators give you one number.  
That number assumes a smooth real return every year.  
Markets don’t work that way.  
(A short thread on what the coast number *is* — and isn’t.)

2/ Coast FIRE formula (real terms):  
coast ≈ full FIRE ÷ (1 + r)^n  
Example: $1.5M target, 5% real, 20 years → ~$565k today.  
If you hold ~$565k invested and never contribute again, the *model* says you may hit full FIRE by traditional retirement age.

3/ What it is NOT:  
• Early retirement tomorrow  
• A healthcare plan  
• A guarantee that returns arrive in a friendly order  

Sequence-of-returns risk still matters — especially if you later withdraw early.

4/ Stress-test the return before you celebrate.  
Same $1.5M target, 20 years:  
• 5% real → ~$565k coast  
• 4% real → ~$684k coast  
Lower r → higher bar. Conservative r is a feature, not pessimism.

5/ Barista FIRE is a different question:  
Portfolio funds the *gap* after work income.  
gap = max(0, spend − work income)  
barista number = gap ÷ withdrawal rate  

Coast = stop contributing. Barista = partial work covers part of spend.

6/ Run both with the same spending and SWR:  
Coast: retirefire.net/calculators/coast-fire  
Barista: retirefire.net/calculators/barista-fire  
Formulas: /methodology  
Educational only — not advice.

### Post 2 — Single tip (X + LI)

```
The 4% rule is a research starting point, not a warranty.

At $60k spending:
• 4.0% → $1.5M
• 3.5% → ~$1.71M
• 3.0% → $2.0M

Early retirees often stress-test below 4% because horizons are longer and sequence risk bites harder.

Tool: retirefire.net/calculators/fire-number
```

### Post 3 — Coast myth

```
“I hit Coast FIRE, so I can retire.”

Usually false.

Coast FIRE means growth *might* finish the job by a traditional retirement age if you stop contributing — not that you can stop working today.

Different question → different calculator.
```

### Post 4 — Barista worked example

```
Spend $60k. Earn $20k part-time. SWR 4%.

Gap = $40k
Barista number = $40k ÷ 0.04 = $1.0M
Full FIRE = $1.5M

That’s a $500k difference — if the work income is realistic and benefits don’t cliff.

Calculator: retirefire.net/calculators/barista-fire
```
