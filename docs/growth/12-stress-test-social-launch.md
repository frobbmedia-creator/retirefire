## X thread (pin-worthy) — 7 tweets

1/ We shipped a free sequence-of-returns stress test on RetireFire.

Not a crystal ball.  
Not “your chance of retiring.”  
An educational range next to the deterministic Coast / Years math.

2/ The main calculators still assume a smooth real return.

Markets don’t.

The stress test runs 1,000 paths with the same average return + fixed volatility, then reports:
• success rate vs target  
• p10 / median / p90 terminal wealth  
• worst / median / best sample paths

3/ Success = terminal portfolio ≥ target under a toy model.

Coast: zero contributions over the horizon → full FIRE target.  
Years: keep contributions → FIRE number target.

If that definition doesn’t match your life, the % is the wrong lens.

4/ Volatility presets (illustrative, not your portfolio):  
σ 12% · 15% · 18%

Mean return = the same shared assumption as the main tools.

Model: r = mean + σ·Z each year (Z ~ normal), seed 42 for reproducibility.

5/ What it is NOT  
• Historical backtesting (cFIREsim-style cycles)  
• Tax-aware  
• A guarantee  
• A reason to YOLO spending the day “success rate” looks high

Details: retirefire.net/methodology

6/ Core calculators stay free.

## Reddit comment kit (no spam)

**When someone asks “is my coast number safe?”**  
> Coast numbers are usually constant-return discounts. Try the same target at 4% and 5% real, and if you use a sim, check how “success” is defined. I’m biased (I work on RetireFire) but the free stress test on the Coast page is explicit about being educational, not a forecast.

**When someone confuses Coast and Barista**  
> Coast ≈ stop contributing; portfolio may reach full FIRE later. Barista ≈ portfolio only covers spending minus work income. Different questions, different targets.
